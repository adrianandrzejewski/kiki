    center-block

    @mixin center-block() { ... }

    Description

    Quickly center any element within its parent. Requires width or max-width to be set

    Parameters

    None.

    Example

    Usage

    .element {
  @include center-block;
}

    CSS Output

    .element {
  display: block;
  margin-right: auto;
  margin-left: auto;
}

    placeholder

    @mixin placeholder() { ... }

    Description

    Mixin to style placeholders. Use it with Autoprefixer.

    Parameters

    None.

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Example

    Usage

    .element {
  @include placeholder {
    color: $eee;
  }
}

    CSS Output (with Autoprefixer)

    .element::-webkit-input-placeholder {
  color: #eee;
}

.element::-moz-placeholder {
  color: #eee;
}

.element:-ms-input-placeholder {
  color: #eee;
}

.element::placeholder {
  color: #eee;
}

    Description

    Provides a quick method for setting an element’s position.

    A CSS position value

    		Position—none
    $args

    List of properties

}

  @include absolute(top 1em right);

.i {

}

  position: absolute;

  position: absolute;

  position: absolute;
  top: 0;
  right: 0;
  bottom: 0;
}

.e {
  position: absolute;
  top: 0;
  right: 0;
  bottom: 0;
  left: 0;
}

.f {
  position: absolute;
  top: 0;
  right: 1em;
}

.g {
  position: absolute;
  top: 1em;
  right: 0;

    size

    @mixin size($width, $height: false) { ... }

    Description

    Sets the width and height of the element.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $width

    Width of element.

    		Css size value—none
    $height

    Height of element.

    If there is no value for height, width value is used for both width and height.

    		Css size valuefalse

    Example

    Usage

    .first-element {
  @include size(2em);
}

.second-element {
  @include size(auto, 10em);
}

    CSS Output

    .first-element {
  width: 2em;
  height: 2em;
}

.second-element {
  width: auto;
  height: 10em;
}

    triangle

    @mixin triangle($size, $color, $direction) { ... }

    Description

    Creates a visual triangle.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $size

    The $size argument can take one or two values—width height.

    		Number or Number number—none
    $color

    The $color argument can take one or two values—foreground-color background-color.

    		Color or Color color—none
    $direction

    The $color argument can take one or two values—foreground-color background-color.

    		Up or Down or Left or Right or Up-right or Up-left or Down-right or Down-left—none

    Example

    Usage

    .triangle-1 { 
  @include triangle(20px 10px, red, down);  
}

.triangle-2 { 
  @include triangle(20px, red yellow, up-right);   
}

    CSS Output

    .triangle-1 {
  height: 0;
  width: 0;
  border-top: 10px solid red;
  border-right: 10px solid transparent;
  border-left: 10px solid transparent;
}

.triangle-2 {
  height: 0;
  width: 0;
  border-top: 20px solid red;
  border-left: 20px solid yellow;
}
